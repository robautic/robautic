## Sobre mim

Desenvolvedora backend com experiência real em suporte técnico N2  o que me dá algo raro: entendo o problema antes de escrever a primeira linha de código.

Construo APIs com Node.js, TypeScript, Fastify e PostgreSQL. Já tenho projetos em produção, testes automatizados e sistemas que resolvem problemas que eu vi acontecer de verdade no ambiente B2B.

Meu interesse de longo prazo é trabalhar com sistemas que integram IA  não como usuária de ferramentas, mas como engenheira que entende os riscos, revisa o código gerado e decide quando a IA está errada.
